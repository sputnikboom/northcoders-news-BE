Api endpoints

  • GET /api -- returns this home page.

  • GET /api/topics -- returns all topics.
  • GET /api/topics/:topic_slug/articles -- returns all articles for a specific topic.
  • POST /api/topics/:topic_slug/articles -- adds a new article to a topic. Requires a JSON object containing both 'title' and 'body' keys.

  • GET /api/articles -- returns all articles.
  • GET /api/articles/:article_id -- returns a specific article.
  • PATCH /api/articles/:article_id -- increments or decrements the vote score of an article by one. Requires a vote query of either 'up' or 'down'.
  • GET /api/articles/:article_id/comments -- returns all of the comments for a specific article.
  • POST /api/articles/:article_id/comments -- adds a new article to a comment. Requires a JSON object contain 'body' and 'created_by' keys.

  • PATCH /api/comments/:comment_id -- increments or decrements the votes of a comment by one. Required a vote query of "up" or "down".
  • DELETE /api/comments/:comment_id -- deletes a specific comment.

  • GET /api/users/:username -- returns the profile data for a specific user.
